Lets your users make the WordPress admin dashboard darker with one click.

== Description ==

This is a beta plugin Please don't use this on production websites unless you're happy with the stability as some things may still be experimental.

Using technology at night time can have a negative effect on your eyesight. Dark Mode will darken the colors of your admin dashboard with just click making it easier for you to work at night or low light and if you do not like just click again to remove dark mode.

== Installation ==

1. Upload the plugin package to the plugins directory.
2. Login to the dashboard and activate the plugin.
3. Click Button on top left corner and click agian remove dark mode

== Frequently Asked Questions ==

= What does this plugin do? =

Click "Dark Mode" Button on top left corner and all Wordpress admin become darker better for lowlight or night time
if you don not like  just click agian remove dark mode it is just one click away. It also work on page reload
